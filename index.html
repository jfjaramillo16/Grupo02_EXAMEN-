Slider - Grupo 02

1. TEORÍA
¿Qué es un Slider (Carrusel)?

Un slider (o carrusel) muestra una serie de elementos (imágenes, texto o video) alternándolos dentro del mismo espacio. Sirve para destacar contenido clave sin recargar la interfaz y mejora la experiencia al presentar información de forma visual y dinámica.

¿Cuándo usarlo?
  • Portadas o hero sections con campañas, productos o mensajes principales.
  • Galerías de imágenes con navegación manual y/o automática.
  • Testimonios o highlights de proyectos/servicios.
Anatomía del Slider en Bootstrap 5

Estructura mínima: contenedor principal, área interna y al menos un ítem activo:

<div id="miSlider" class="carousel slide" data-bs-ride="carousel">
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="img/imagen1.jpg" class="d-block w-100" alt="Descripción de la imagen 1">
    </div>
  </div>
</div>
  • .carousel: contenedor principal del carrusel.
  • .carousel-inner: envoltorio de todas las diapositivas.
  • .carousel-item: cada diapositiva (la primera con .active).
  • <img class="d-block w-100">: imagen responsiva a ancho completo.
Navegación: indicadores y controles

Los indicadores (puntos) y los controles (flechas) permiten moverse entre diapositivas:

<!-- Indicadores -->
<div class="carousel-indicators">
  <button type="button" data-bs-target="#miSlider" data-bs-slide-to="0" class="active"></button>
  <button type="button" data-bs-target="#miSlider" data-bs-slide-to="1"></button>
  <button type="button" data-bs-target="#miSlider" data-bs-slide-to="2"></button>
</div>

<!-- Controles -->
<button class="carousel-control-prev" type="button" data-bs-target="#miSlider" data-bs-slide="prev">
  <span class="carousel-control-prev-icon" aria-hidden="true"></span>
  <span class="visually-hidden">Anterior</span>
</button>
<button class="carousel-control-next" type="button" data-bs-target="#miSlider" data-bs-slide="next">
  <span class="carousel-control-next-icon" aria-hidden="true"></span>
  <span class="visually-hidden">Siguiente</span>
</button>
Captions (títulos y textos sobre la imagen)

Los captions explican cada imagen. Van dentro de .carousel-item en un contenedor .carousel-caption. En tu style.css ya usas estilos que contrastan bien con el fondo. 

<div class="carousel-item active">
  <img src="img/sala.jpg" class="d-block w-100" alt="Sala">
  <div class="carousel-caption d-none d-md-block">
    <h5>Sala</h5>
    <p>Ambiente moderno y acogedor.</p>
  </div>
</div>
Atributos clave (data-*)
  • data-bs-ride="carousel": ciclo automático.
  • data-bs-interval="3500": tiempo (ms) entre transiciones.
  • data-bs-pause="hover": pausa al pasar el mouse.
  • data-bs-wrap="true": vuelve al primer ítem al terminar.
  • data-bs-touch="true": gestos táctiles en móviles.
Tipos de transición
  • Deslizamiento (por defecto): horizontal.
  • Desvanecimiento (suave): agrega .carousel-fade al contenedor.
<div id="miSlider" class="carousel slide carousel-fade" data-bs-ride="carousel">...</div>

EJERCICIO: Slider (Fade) – Interfaz Grupo 2

En esta sección se explica, paso a paso, cómo se implementó el slider con efecto de desvanecimiento (fade) dentro de la columna derecha (#rightMenu) de tu interfaz. Al final verás la ejecución en vivo.

Explicación del Código Principal
1. Ubicación en la maqueta (Grid)

El slider se colocó en la columna derecha de la fila principal, ajustando las columnas a 2 + 4 + 6 = 12 para que todo encaje. Ahí vive el contenedor del slider. 

<div class="row">
  <div id="leftMenu"  class="col-lg-2 col-md-2 col-sm-12">...</div>
  <div id="content"   class="col-lg-4 col-md-4 col-sm-12">...</div>

  <!-- AQUÍ va el slider -->
  <div id="rightMenu" class="col-lg-6 col-md-7 col-sm-12 text-center">
    <div id="sliderFade">
      <img src="img/imagen1.jpg" class="fade-img" alt="Sala moderna">
      <img src="img/imagen2.jpg" class="fade-img" alt="Comedor elegante">
      <img src="img/imagen3.jpg" class="fade-img" alt="Perro en el jardín">
    </div>
  </div>
</div>
2. Estructura HTML del Slider

El slider es un contenedor #sliderFade con varias imágenes que comparten la clase .fade-img. La primera se muestra y el resto se ocultan; el JavaScript se encarga de irlas desvaneciendo. 

<div id="sliderFade">
  <img src="img/imagen1.jpg" class="fade-img" alt="Sala moderna">
  <img src="img/imagen2.jpg" class="fade-img" alt="Comedor elegante">
  <img src="img/imagen3.jpg" class="fade-img" alt="Perro en el jardín">
</div>
3. Estilos CSS Claves (Responsive + Recorte)

Para que las imágenes no se salgan del contenedor y mantengan proporción, se usa overflow:hidden, object-fit:cover y una relación de aspecto 16:9. Además, cada foto ocupa todo el contenedor. 

#rightMenu{ overflow:hidden; }

#rightMenu #sliderFade{
  position:relative; width:100%; max-width:100%;
  overflow:hidden; border-radius:8px; background:#000;
  aspect-ratio:16/9; /* proporción */
}

@supports not (aspect-ratio:16/9){
  #rightMenu #sliderFade{ height:52vw; min-height:220px; max-height:480px; }
}

#rightMenu #sliderFade .fade-img{
  position:absolute; inset:0; width:100%; height:100%;
  object-fit:cover; display:none;
}
#rightMenu #sliderFade .fade-img:first-child{ display:block; }
4. Lógica JavaScript (Fade automático)

El script recorre las imágenes cada 3 segundos. Se oculta la actual con fadeOut() y se muestra la siguiente con fadeIn(). Esta función ya está en tu js/main.js. 

function initFadeSlider(root){
  var $slider = root ? $(root) : $('#rightMenu #sliderFade');
  if(!$slider.length || $slider.data('fade-init')) return;

  var $imgs = $slider.find('.fade-img');
  if($imgs.length < 2){ $imgs.show(); $slider.data('fade-init',true); return; }

  $imgs.hide().eq(0).show();
  var i = 0;
  setInterval(function(){
    var next = (i + 1) % $imgs.length;
    $imgs.eq(i).fadeOut(800);
    $imgs.eq(next).fadeIn(800);
    i = next;
  }, 3000);

  $slider.data('fade-init', true);
}
5. Buenas prácticas aplicadas
  • Alt descriptivo en cada imagen (accesibilidad).
  • object-fit: cover para no deformar las fotos.
  • overflow: hidden para que nada se salga del marco.
  • aspect-ratio (y fallback) para mantener la proporción del slider.
